## 二分查找(Binary Search)
二分查找也称折半查找（Binary Search）,是一种效率较高的查找方法,但是二分查找要求线性表必须采用顺序存储结构，而且表中元素按关键字有序排列。
- 时间复杂度为O(h)=O(log2n).

### 算法步骤
1. 将序列从中间分成两块.
2. 对比元素与中间数，如果中间数小于元素，那么将中间数之后的数字分为两块，继续对比查找.
